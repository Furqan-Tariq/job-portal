# Oberland Jobs Homepage Clone

This is a Next.js 14 App Router project reproducing the homepage of oberland-jobs.de with TypeScript and Tailwind CSS.

## Getting Started

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` - Next.js App Router pages and layouts
- `src/components/layout/` - Global layout components (header, footer)
- `src/components/home/` - Homepage-specific components
- `src/components/ui/` - Reusable UI components
- `src/lib/` - Types, mock data, and API functions

## Features

- Fully responsive design
- Modular component architecture
- TypeScript for type safety
- Tailwind CSS with semantic design tokens
- Mock data for jobs, employers, regions, and videos
- YouTube video embeds for job highlights

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18
